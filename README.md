# 貓狗大平台

* 網站功能
    * 領養人：
        * 找到喜歡的動物進⾏收藏
            * 隨機配對 vs 條件配對
        * 建立收藏狗狗貓貓的明細
        * 預約時間去看中途之家的狗狗貓貓
    * 送養人：
        * 在網站上新增動物讓領養者搜尋
        * 管理領養人的預約
